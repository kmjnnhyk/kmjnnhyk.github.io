# 글 작성 스타일 가이드

## 참조

사용자의 Medium 블로그: https://medium.com/@kmjnnhyk

## 문체 특징

### 톤
- **반말 중심의 캐주얼한 서술체**: "~했다", "~인 것 같다", "~해봤는데"
- 1인칭 화자("나", "내가")로 개인적인 경험을 직접 서술
- 독자에게 말을 거는 듯한 의문형 활용: "이게 맞나?", "왜 이렇게 되는 걸까?"
- ";;" 같은 캐주얼한 표현으로 감정을 드러냄

### 구조
- 문제 상황 → 시행착오 → 해결의 자연스러운 흐름
- 짧고 간결한 소제목
- 깨달음의 순간을 강조 ("알고 보니", "결국", "생각보다")

### 설명 방식
- 기술 개념을 자기 말로 풀어서 설명
- "쉽게 말하면", "결국 이건" 같은 재해석 표현
- 코드보다 개념과 판단 근거 중심

### 금지 사항 (AI 말투)
- "~입니다", "~하겠습니다" 같은 경어체 금지
- "먼저 ~을 살펴보겠습니다" 같은 발표 톤 금지
- "이 글에서는 ~에 대해 알아보겠습니다" 같은 블로그 공식 도입부 금지
- "결론적으로", "요약하자면" 같은 정형화된 마무리 금지
- "효과적으로", "체계적으로" 같은 과잉 부사 금지
- 빈 수식어 나열 금지 ("강력하고 유연한 도구")
